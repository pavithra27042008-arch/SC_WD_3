Quiz Game Application

📌Overview

This is a simple Quiz Game Application developed using HTML, CSS, and JavaScript. Users answer multiple-choice questions and receive their final score after completing the quiz.

🚀Features

- Interactive user interface
- Multiple-choice questions
- Instant answer validation
- Score calculation
- Restart quiz functionality
- Responsive design

🛠️Technologies Used

- HTML5
- CSS3
- JavaScript

📂 Folder Structure

Quiz-Game-Application/

- index.html
- style.css
- script.js
- README.md
- assets/

▶️How to Run

1. Download or clone the project.
2. Open `index.html` in any web browser.
3. Click the answers.
4. View your score.
5. Click **Play Again** to restart.
